隐私政策

最后更新日期：2026年3月28日  ·  生效日期：2026年3月28日

⚓ 时光锚承诺：我们不收集、不上传、不出售任何您的个人数据。所有数据仅存储在您的设备本地。

一、我们是谁

时光锚（TimeAnchor）是一款专注力追踪应用，帮助用户设定目标、记录目标期间的应用使用情况，从而了解自己的时间分布。 本应用由个人开发者开发，非商业运营。

二、我们收集哪些数据

时光锚仅在您的设备本地处理以下数据，不会将任何数据发送至外部服务器：

  • 应用使用记录：记录目标监控期间您使用了哪些应用及使用时长（通过无障碍服务获取）
  • 目标信息：您创建的目标标题、预计时长、开始/结束时间
  • App 统计数据：通过 Android 使用情况访问权限（UsageStats API）获取的日常应用使用汇总
  • 标签数据：您为应用自定义的分类标签

以上所有数据均仅存储在您设备的本地数据库中，随应用卸载而消失。

三、我们不收集的数据

  • 不收集您的姓名、手机号、电子邮件等个人身份信息
  • 不读取您的通讯记录、联系人、相机、麦克风
  • 不读取屏幕内容（无障碍服务仅用于检测应用切换事件，不读取任何页面文字或截图）
  • 不收集设备标识符（IMEI、广告 ID 等）
  • 不进行任何网络通信

四、我们使用哪些权限及用途

  • 无障碍服务（AccessibilityService）：仅用于在目标监控期间检测您切换到了哪个应用， 精确记录每个应用的使用时段。不读取任何界面内容。
  • 使用情况访问权限（UsageStats）：用于获取每日应用使用汇总数据， 展示在时间轴和统计页面。
  • INTERNET 权限：本应用无需网络访问， 如有此权限声明仅为 Flutter 框架默认配置，实际不进行任何网络请求。

五、数据共享与披露

我们不与任何第三方共享您的数据。时光锚不集成任何广告 SDK、分析 SDK 或第三方数据服务。

六、数据安全

所有数据存储在设备本地的 SQLite 数据库中，受 Android 系统的应用沙箱保护，其他应用无法访问。 您可以随时通过「设置 → 导出数据」导出您的数据，或卸载应用以删除全部数据。

七、儿童隐私

时光锚不专门面向13岁以下儿童，也不会故意收集儿童的个人信息。

八、隐私政策的变更

如我们对本政策进行重大修改，将通过应用内通知或更新本页面告知您。 继续使用本应用视为接受修订后的政策。
